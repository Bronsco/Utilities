Demos

LMN Tool

In order to access the demos of the Local Model Networks Toolbox from the command window type LMNTrainDemo.

  • LMNDemo1 - Static process with 1 input and 1 output
  • LMNDemo2 - Static process with 2 inputs and 1 output
  • LMNDemo3 - Static process with 2 inputs and 1 output
  • LMNDemo4 - Static process with 2 inputs and 2 outputs

LOLIMOT

In order to access the demos for LOLIMOT from the command window type lolimotDemo.

  • lolimotDemo1 - Static process with 1 input and 1 output
  • lolimotDemo2 - Static process with 2 inputs and 1 output
  • lolimotDemo3 - Static process with 2 inputs and 2 outputs
  • lolimotDemo4 - Dynamic process of second order with 1 input and 1 output
  • lolimotDemo5 - Dynamic process of first order with 2 inputs and 2 outputs
  • lolimotDemo6 - Estimate a model given an initial structure

HILOMOT

In order to access the demos for HILOMOT from the command window type hilomotDemo.

  • hilomotDemo1 - Static process with 1 input and 1 output
  • hilomotDemo2 - Static process with 2 inputs and 1 output
  • hilomotDemo3 - Static process with 2 inputs and 2 outputs
  • hilomotDemo4 - Static process: Global estimation and ridge regression after training
  • hilomotDemo5 - Static process: Backward elimination after training
  • hilomotDemo6 - Dynamic process of second order with 1 input and 1 output
  • hilomotDemo7 - Dynamic process of first order with 2 inputs and 2 outputs
  • hilomotDemo8 - Dynamic prosess: Parameter estimation in NARX, IV and NOE setting
